## Install

```bash
pip install concrete-pump-cpa-mcp
```

## Tools

| Tool | Purpose |
|------|---------|
| `check_cpa_compliance` | CPA Concrete Pumping Group safety standards check |
| `operator_cert_check` | Concrete pump operator certification verification |
| `calculate_ground_bearing` | BS EN 12001 ground bearing capacity calculator |
| `exclusion_zone_check` | Exclusion zone setup per pump type and reach |
| `pre_pour_checklist` | Pre-pour safety + setup checklist |
